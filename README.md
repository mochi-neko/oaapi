# oaapi
An unofficial Rust client for OpenAI API.

## Installation

Run the following Cargo command in your project directory:

```shell
cargo add oaapi
```

or add the following line to your Cargo.toml:

```toml
[dependencies]
oaapi = "0.1.0"
```
